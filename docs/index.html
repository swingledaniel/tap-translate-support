Getting Help

If something is not working, please open a public GitHub issue with your iPhone model, iOS version, app version, and a short description of what happened.

Do not include private screenshots, personal text, API keys, or account details in public issues.

Shortcut Setup

Tap Translate works best with an iOS Shortcut that takes a screenshot and passes it to the app. Create a Shortcut with these actions:

  1. Take Screenshot.
  2. Translate Screenshot, using the screenshot from the previous action as input.

You can trigger that Shortcut from Back Tap or AssistiveTouch in the iOS Settings app.

Common Checks

  • Make sure the Shortcut passes the screenshot image into Translate Screenshot.
  • If cloud translation is unavailable, try System Translation or check your network connection.
  • If text boxes are missing, try a clearer screenshot with less motion blur or lower compression.
